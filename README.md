* Đã sử dụng variable cho các màu
* Đã sử dụng mixin để responsive
* Đã sử dụng extend cho các CSS trùng lặp như flexbox
* Đã sử dụng vòng lặp khi CSS cho nhiều Icon
## BEM

## Animation, transition
